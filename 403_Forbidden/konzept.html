Konzept

Das erste Konzept war, mit Hilfe einer API eine Einkaufsmöglichkeit für den Nutzer zu ermöglichen. Dabei standen mehrere API's zur Auswahl, welche aber aufgrund fehlender Kenntnisse zu schwer zu implentieren erschienen. Deshalb wurde als Gruppe eine zweite Idee überlegt, welche einfacher realisierbar und trotzdem nutzvoll für den Endverbraucher ist.

Das Konzept für die WBA - Challenge ist eine Energieverbrauchsanzeige, über die einzelnen Bauteile im Tesla, welche durch Graphen dargestellt werden. Hierfür wird die D3 API genutzt, um die grafische Darstellung der Daten zu ermöglichen. Als Grundstein dabei, nutzt die Gruppe das bereitgestellte Mockup und Design. Dieses wird über den Tag hinaus so bearbeitet werden, das die Idee darin anschaubar eingebettet wird.

Das Konzept soll nachfolgende Eigenschaften enthalten:
- Dem Nutzer wird die Möglichkeit gegeben, einzelne Verbraucher mithilfe von Optionen zu sortieren und filtern
- Der Verbrauch wird in einem festem Zeitintervall aktualisiert
- Die grafische Darstellung wird frei auswählbar sein
- Als Livedemo werden zufällig generierte Parameter zur Anschauung des Designs genutzt
